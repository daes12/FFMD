Bases de Datos

Declaración

La clase para la Base de Datos se carga automáticamente y está disponible para su trabajo con los Modelos.

Realizar Consulta

//Consulta que obtiene resultados los cuales se reciben en forma de array
$Resultado_Consulta = $this->bd->consulta('select * from tiendas');

$this->bd->consulta('delete from oficina where id_oficina = 12');

$this->bd->consulta('insert into departamento values(NULL, 1, "MiTienda")');

Saber el total de filas devueltas por un select es tan facil como: 
$Filas = count($Resultado_Consulta);

Limpieza de variables

Una función fundamental es la de limpiar las variables que se agregarán a una consulta.
En FFMD se realiza de la siguiente manera: 

$Variable = "<script>alert('Péz')</script>";
$Variable = $this->bd->limpieza_variable($Variable);
echo $Variable;//&lt;script&gt;alert(&#039;P&eacute;z&#039;)&lt;/script&gt;
