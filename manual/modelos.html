Cargar Modelo

Para cargar un Modelo se hace uso de la "vieja escuela" (es la mejor frase que se me ocurrió): 

//Se incluye el modelo
include('aplicacion/modelo/subdirectorio/su_modelo.php');

//Se crea el objeto
$Modelo = new Una_clase();

//Se ejecuta el método con las acciones requeridas
$Modelo->ejecutar_metodo();

Departamento de quejas:
¿Por qué no existe una función para evitar hacer un include y poner la ruta completa?
Recuerde que el núcleo de FFMD contiene solamente una poca cantidad de clases y funciones para ganar en velocidad y bajo consumo de recursos.
