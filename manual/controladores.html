Controladores

Estructura

Archivo ejemplo.php 

<? if(!defined('SIST_ACTIVO')){ exit(); }
//Evitamos que el script se ejecute fuera del Framework

//Se heredan los objetos necesarios para el controlador
class ejemplo extends ffmd_controlador
{
  
  //Método principal que recibe las variables contenidas en la URL
  public function ejemplo_c($Variables)
  {
    
    //Acciones
    //Carga de los Modelos
    //Carga de las Vistas
    
  }
  
}
Nota 1: La clase debe poseer el mismo nombre que el archivo controlador.
Nota 2: Es necesario nombrar el método principal igual a la clase y agregarle el sufijo _c De lo contrario se obtendrá una nota de error.
