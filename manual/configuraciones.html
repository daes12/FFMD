Configuraciones

El archivo encargado de almacenar las configuraciones del sistema es "configuraciones.conf.php" y está ubicado en el directorio "configuracion".

Bases de Datos

Configuraciones utilizadas por la clase encargada de conectar y trabajar con la base de datos (mysql). 

/**
  *Configuraciones para la Base de datos
*/
$this->BD['servidor'] = 'localhost';
$this->BD['usuario'] = 'root';
$this->BD['contrasenha'] = '';
$this->BD['base_datos'] = '';

Modelos Automáticos

Para dejar que el sistema cargue automáticamente alguno de los modelos que pertenecen a nuestra aplicación; debemos agregarlos como una lista de array, p.ej.: 

/**
  *Modelos que se cargan automaticamente en el sistema.
  *array(Nombre_Clase => Ubicacion_Modelo)
*/
$this->Auto_Modelos = array(
  'ejemplo' => 'carpeta/ejemplo'
);
